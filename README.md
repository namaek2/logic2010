script for installing Logic 2010 on Linux, wayland. 

Usage:
   ./install_logic2010.sh [options] [path-to-zip]

Options:
    --app                 Install into ~/Applications as a double-clickable app(instead of the default ./logic2010 + run script).
    --prefix DIR          Install directory (default: ./logic2010).
    --no-verify           Skip the checksum check.
    -y, --yes             Don't prompt; assume "yes".
    -h, --help            Show this help.
